# Algorithmable

Useful algorithms for everyday usage implemented using Ruby.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'algorithmable'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install algorithmable
